# Coca-Cola Animation

A creative HTML & CSS project that brings the iconic Coca-Cola can to life with smooth, interactive animations. On hover, the Coke can animates with a refreshing effect, demonstrating modern CSS techniques without any JavaScript.

## Features

- Pure HTML & CSS implementation  
- Interactive hover animation  
- Smooth transitions and visual effects  
- Lightweight and easy to integrate  

## Demo

Hover over the Coca-Cola can to see the animation in action.
